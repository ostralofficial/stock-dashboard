# 주식 재무 분석 대시보드

## 설치 및 실행

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 기존 종목 일괄 등록 (최초 1회)
EPS-ACCUM.xlsx를 같은 폴더에 놓고 실행:
```bash
python import_stocks.py --file EPS-ACCUM.xlsx
```

### 3. 앱 실행
```bash
streamlit run app.py
```
브라우저에서 http://localhost:8501 열림

---

## 사용 순서

1. **설정** → DART API 키 입력
2. **종목 관리** → 종목 확인 / 추가 / 삭제
3. **데이터 수집** → 연도 범위 선택 후 수집 시작
   - 하루 10,000회 제한 → 200종목씩 나눠서 3일에 걸쳐 수집
4. **대시보드** → 종목/항목/연도 선택 → 그래프 확인
5. **수동 입력** → 해외매출, 수주잔고 등 DART에 없는 항목 입력

## 데이터 수집 전략 (606종목 × 12년)

| 일차 | 수집 종목 | 예상 호출 수 |
|------|-----------|------------|
| 1일차 | 1~200번   | ~7,200회   |
| 2일차 | 201~400번 | ~7,200회   |
| 3일차 | 401~606번 | ~7,400회   |

이후 매년 분기 결산 후 → 606 × 4회 = 2,424회 (하루 가능)

## 파일 구조
```
stock_app/
├── app.py              # 메인 대시보드
├── db.py               # DB 관리
├── dart_collector.py   # DART API 수집
├── import_stocks.py    # 기존 엑셀 종목 임포트
├── requirements.txt    # 패키지 목록
├── stock_data.db       # SQLite DB (자동 생성)
└── config.json         # API 키 설정 (자동 생성)
```
